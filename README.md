# Divi Semantic Fix

Adds `role="main"` to Divi's `<div id="main-content">` element to improve accessibility.  
This ensures the main content landmark is recognized by screen readers and accessibility tools.

## Installation

1. Upload `divi-semantic-fix.php` to `/wp-content/plugins/`.
2. Activate the plugin via the WordPress admin dashboard.
3. Works automatically â€” no settings required.

## Requirements

- WordPress  
- Divi theme (plugin checks for Divi before running)
